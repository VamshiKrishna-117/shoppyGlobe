# ShoppyGlobe E-commerce

A modern, responsive e-commerce web application built with React, Vite, and Redux Toolkit.

## GitHub Repository
[https://github.com/VamshiKrishna-117/shoppyGlobe.git]

## Features
- Dynamic routing with React Router (`createBrowserRouter`)
- Code splitting and lazy loading components (`React.lazy` & `Suspense`)
- Global state management for Cart & Search using Redux Toolkit
- Custom hook for API data fetching (`useEffect`)
- Responsive design and CSS styling

## Tech Stack
- React 18+
- Vite
- React Router DOM 
- Redux Toolkit & React-Redux

## How to Run Locally
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
